argument와 parameter

PARAMETER란
매개변수란 뜻이다. 즉, 함수를 정의할때 외부로부터 받아들이는 임의의 값을 의미한다.

예)
function f(x,y){
	return X+Y;
};

에서 x,y가 파라미터라고 할 수 있다.


그럼 Argument란

우리말로는 '인수'라고 번역되는데,
함수를 호출할 때 이 때 사용하게 되는 일련의 값들을 argument라고 부른다.
위에 파라미터의 예를 들었던 함수를 호출한다고 하면

f(3,4);
에서 3,4등이 argument이다.

즉, 파라미터의 값으로 argument 3과 4를 대입했다.
라는 의미가 성립하는 것이다.

이처럼 파라미터와 아규먼트는 분명 같은 위치에 있지만 다른 의미로 쓰이는 것을 알 수 있다.
